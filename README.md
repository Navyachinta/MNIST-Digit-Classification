# MNIST Digit Classification

This project builds a simple Convolutional Neural Network (CNN) using TensorFlow/Keras to classify handwritten digits from the MNIST dataset.

## Model Summary
- Convolutional Layer
- Max Pooling
- Dense Layers
- Softmax for final prediction
